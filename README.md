# Card Memory Game
메모리 카드 게임은 기억력과 집중력을 향상시키기 위해 많이 사용되는 놀이로 이 프로젝트에서는 트럼프 카드를 사용합니다. 게임의 목표는 뒤집혀 있는 카드들 중에서 같은 카드를 찾아 모든 스테이지를 클리어해야 합니다.

stage는 총 3단계로 구성되어 있으며 2x2,2x4,3x6의 카드들을 30,60,90초 이내에 맞춰야 클리어할 수 있습니다.
